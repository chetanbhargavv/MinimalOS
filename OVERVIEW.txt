BOOT PROCESS
═══════════════════════════════════════════════════════════════════════════════

Stage 1: BIOS Boot (Real Mode - 16-bit)
    ↓
    • BIOS loads boot sector to 0x7C00
    • Initialize segments and stack
    • Load kernel from disk (15 sectors)
    • Enable A20 line
    ↓
Stage 2: Protected Mode (32-bit)
    ↓
    • Load GDT
    • Set CR0.PE bit
    • Setup paging structures
    ↓
Stage 3: Long Mode (64-bit)
    ↓
    • Enable PAE (CR4.PAE)
    • Set EFER.LME
    • Enable paging (CR0.PG)
    • Load 64-bit GDT
    ↓
Stage 4: Kernel Initialization
    ↓
    • Setup kernel stack
    • Initialize screen
    • Initialize memory
    • Initialize scheduler
    • Start shell
